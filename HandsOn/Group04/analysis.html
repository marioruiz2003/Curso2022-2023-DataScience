Analising dataset

Columna Tipo Descripción
TIPO DE CONTENEDOR String Indica el tipo de contenedor(punto limpio,ropa,pilas,etc)
DIRECCIÓN String Indica donde se encuentra ubicado (calle/paseo) cada contendor
DISTRITO String Indica el distrito donde se encuentra cada contenedor
COORDENADA X Float Indica la respectiva coordenda x de cada contenedor
COORDENADA Y Float Indica la respectiva coordenda y de cada contenedor
HORARIO String Indica el horario de disponibilidad de cada contenedor
CÓDIGO POSTAL Int Indica el código postal donde se ubica el contenedor
